# Portfolio (2016)

Working repository for a personal website refresh

### Requirements

[Sass](http://http://sass-lang.com/)
```
gem install sass
```

[Bourbon](http://bourbon.io/)
```
gem install bourbon
```

### Installation

Install Bourbon into `scss` directory
```
bourbon install
```

Compile SCSS
```
sass --watch scss:css
```
